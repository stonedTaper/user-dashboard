# User Dashboard
================

## Description
---------------

User Dashboard is a web application designed to provide users with a personalized interface for monitoring and managing their account information. This dashboard allows users to view their profile details, track their account activity, and access relevant features and settings.

## Features
------------

* **Profile Management**:
	+ View and edit user profile information
	+ Update password and security settings
* **Account Activity**:
	+ View recent login history
	+ Track account balance and transaction history
* **Settings**:
	+ Customize dashboard layout and preferences
	+ Enable/disable notifications and two-factor authentication

## Technologies Used
-------------------

### Frontend

* **React**: For building the user interface components
* **Redux**: For state management and global state storage
* **Axios**: For making HTTP requests to the backend API

### Backend

* **Node.js**: For server-side logic and API handling
* **Express**: As the web framework for building the API
* **MongoDB**: As the database for storing user data

### Other

* **Webpack**: For bundling and managing frontend assets
* **ESLint**: For code linting and quality assurance

## Installation
------------

### Prerequisites

* Node.js (version 14 or higher)
* npm (version 6 or higher)
* MongoDB (version 4 or higher)

### Installation Steps

1. Clone the repository using `git clone https://github.com/your-username/user-dashboard.git`
2. Navigate to the project directory using `cd user-dashboard`
3. Install dependencies using `npm install`
4. Start the frontend server using `npm start`
5. Start the backend server using `node server.js`
6. Access the application at `http://localhost:3000` (frontend) or `http://localhost:8080` (backend)
